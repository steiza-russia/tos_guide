# Как оценить ремонтные работы по контейнеру?

## Находим поврежденный контейнер

1. Заходим на страницу "Контейнеры"
2. Находим нужный контейнер по его номеру

Если контейнер поврежден, то в колонке "Поврежден" будет статус акта ремонта. Если нажать на эту ссылку, то произойдет переход на акт ремонта

> У поврежденных контейнеров автоматически создается акт ремонта после приемки

> Пока акт не в статусе "Выполнен" или "Отклонен", контейнер недоступен для выдачи

![Список клиентов](/images/38.png)

## Переходим во вкладку "Ремонтные работы"

Внутри акта ремонта переходим во вкладку "Ремонтные работы" для оценки ремонта

![Список клиентов](/images/41.png)

На этой вкладке изображены все повреждения, которые были добавлены на этапе приемки

> Если на этапе оценки или ремонта вы обнаружили новые повреждения, то их можно добавить прямо из акта ремонта по кнопке "Добавить повреждение"

1. Общая стоимость отображается с учетом крановых операций по уборке и постановке контейнера в ремонтную зону
2. Пока хотя бы одна ремонтная группа в статусе "Требует оценки", то весь акт будет в статусе "Создан"

![Список клиентов](/images/42.png)

## Оцениваем ремонт

Нажимаем на одну из ремонтных групп в статусе "Требует оценки". Попадаем на страницу с выбором услуг, которые будут оказаны в рамках работ

Нажимаем на плюсик, чтобы добавить услугу

![Список клиентов](/images/43.png)

Выбираем услуги из списка. Можно искать по наименованию

![Список клиентов](/images/44.png)

Цены по услугам можно изменить отредактировав соответствующие поля. Также отображается общая стоимость по ремонтной группе в детализации по работам и материалам. После завершения добавления услуг важно нажать "Сохранить"

![Список клиентов](/images/45.png)

После этого статус ремонтной группы изменится на "Оценен"

Если больше нет ремонтных групп в статусе "Требует оценки", то весь акт автоматически перейдет в статус "Требует подтверждения". После этого клиент сможет зайти в личный кабинет и подтвердить или отклонить оценку

![Список клиентов](/images/46.png)
