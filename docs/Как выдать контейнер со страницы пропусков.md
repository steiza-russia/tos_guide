# Выдаем контейнер со страницы пропусков (релизов)

## Находим нужный пропуск (релиз)

Переходим на страницу пропусков (релизов). Пункт в меню "Пропуска". И находим нужный пропуск. Найти его можно по

1. Номеру контейнера (если выдаем по номерной заявке)
2. По номеру пропуска
3. По клиенту
4. По букингу

Нажимаем на нужный пропуск, чтобы перейти в его карточку

![Находим контейнер](/images/31.png)

## Впускаем машину

В карточке пропуска нажимаем "Впустить машину". Это действие меняет статус пропуска

> Это действие не обратимо и фиксирует въезд машины на терминал

После этого появится индикатор количества времени на терминале. Он будет отображаться до тех пор, пока машина не выйдет с терминала

![Находим контейнер](/images/32.png)

## Выдаем контейнер

Нажимаем кнопку "Выдать контейнер". Система автоматически перейдет на страницу создания акта выдачи и предложит выбрать конетйнер. Выбираем

> Будьте внимательны, это действие автоматически создает черновик акта выдачи

![Находим контейнер](/images/33.png)

## Вносим информацию о выдаче (Если нужно)

Вносим информацию о выдаче (автомобиль, водитель, контейнер) и нажимаем "Сохранить"

![Находим контейнер](/images/34.png)

## Формируем акт выдачи

Нажимаем на кнопку "Сформировать акт"

> Это действие не обратимо. После этого все услуги считаются завершенными, а контейнер автоматически вывозится с терминала

![Находим контейнер](/images/35.png)

## Печатаем акт

После формирования акта можно распечатать акт

![Находим контейнер](/images/36.png)
