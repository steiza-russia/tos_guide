# Как отклонить ремонт?

## Находим поврежденный контейнер

1. Заходим на страницу "Контейнеры"
2. Находим нужный контейнер по его номеру

Если контейнер поврежден, то в колонке "Поврежден" будет статус акта ремонта

> У поврежденных контейнеров автоматически создается акт ремонта после приемки

![Список клиентов](/images/47.png)

## Отклоняем ремонт

1. Нажимаем на контейнер, чтобы перейти в его карточку
2. Выбираем "Действия" и нажимаем "Отклонить ремонт"

![Список клиентов](/images/39.png)

> Пока у поврежденного контейнера акт не в статусе "Выполнен" или "Отклонен", контейнер недоступен для выдачи
