# Как добавить клиента в систему?

## Добавление клиента

Выбрать пункт слева в меню «Клиенты». Откроется страница списка клиентов. Нажать кнопку «Добавить клиента» 

![Список клиентов](/images/1.png)

Откроется страница заполнения информация о клиенте

1. ФИО и название можно заполнять одним значением в случае с клиентами юрлицами
2. Пароль нужно запомнить и передать клиенту. Он будет заходить в систему по этому паролю. Пароль можно изменить в любой момент
3. По email клиент будет тоже в ходить в систему
4. Обязательно нужно выбрать терминал, к которому будет привязан этот клиент
5. Нажать «сохранить». Клиент будет создан и появится в общем списке 

![Форма создания клиента](/images/2.png)

## Добавление договора

Клиенту можно добавить в системе договор. Это не обязательно, но удобно. Перейти во вкладку «Документы». Нажать «Добавить документ», заполнить форму и нажать «Загрузить». Договор появится в списке и его можно скачать в любой момент 

![Добавление договора](/images/3.png)

## Добавление стока

Чтобы добавить сток клиенту, нужно перейти во вкладку «Стоки». Здесь отображается список стоков клиента. Нажать на кнопку «Добавить сток». Заполнить: 

1. Терминал, на котором будет доступен сток
2. Название стока
3. «Доступно для номерной выдачи» - дает возможность создавать номерную заявку на выдачу. Если галочку не поставить, то из стока можно будет вывезти только по безмерной заявке. Эту галочку можно изменить в любой момент

![Добавление стока](/images/5.png)
